Step 1 of 4 Introduction

Why connect GitHub?

GitHub integration enables you to:

  • Browse and switch to any branch from your repositories
  • Connect local repos to different GitHub repositories
  • Access private repositories you have permission to

Note: This setup creates a GitHub OAuth App in your account. Your credentials are stored locally and never sent to any third-party server.

Create a GitHub OAuth App

Follow these steps to create your OAuth application:

  1. 1

    Go to GitHub Developer Settings

    Open GitHub Developer Settings
  2. 2

    Click "New OAuth App" button

  3. 3

    Fill in the form with these values:

    Application name: Local Stack Admin
    Homepage URL: http://127.0.0.1:9010
    Authorization callback URL: http://127.0.0.1:9010/oauth/callback
  4. 4

    Click "Register application"

After creating the app, you'll see the Client ID. Click "Generate a new client secret" to get the Client Secret.

Enter Client ID

Copy the Client ID from your GitHub OAuth App settings page.

Please enter a valid Client ID

Where to find it: On your OAuth App page, the Client ID is displayed at the top of the page.

Enter Client Secret

Generate and copy the Client Secret from your GitHub OAuth App.

Please enter a valid Client Secret

Important: The Client Secret is only shown once when generated. Make sure to copy it before closing the page. Your secret is stored locally in ~/.run_local_stack_config.json.
